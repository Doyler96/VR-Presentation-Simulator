How to Open Folder in Unity

1. Unzip the Folder
2. Open up Unity
3. Click on Open in the top right hand corner
4. Find the folder that you just unzipped
5. It should say 'Select Folder'
6. Then it should open in Unity

How to Set the Oculus up
